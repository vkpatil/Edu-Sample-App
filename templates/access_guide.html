Public vs Private

Public Pages

Home, Access Guide, Sign In, and Student Registration are available without login.

Private Pages

Dashboard, Courses, Enrollments, Quizzes, Profile, and User Management require authentication and role checks.

Role Access Matrix

Area Student Teacher Admin Super Admin
Dashboard Own dashboard Own dashboard Admin dashboard Global dashboard
Courses Browse View own courses View all View all
Create Course No Yes Yes Yes
Enrollments Own records Own course records All All
Quizzes Published + enrolled Own course quizzes All All
User Management No No Yes Yes

If You See Permission Errors

  • Confirm you are logged in with the expected account.
  • Check your role badge in the app top bar.
  • Use the in-app Help page after login for detailed role actions.

Need an Access Change?

Students can sign up directly. Staff access changes should be requested from Admin or Super Admin.
